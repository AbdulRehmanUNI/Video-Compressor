# Video-Compress
 
## Getting Started

### Installation
1. Clone the repository.
2. Navigate to the project directory:
```
cd video-compressor
```
3. Install the dependencies:
```
npm install
```

### Running the App
1. Start the development server:
```
npm run dev
```
2. Open your browser and navigate to `http://localhost:5173` to see the app running.
